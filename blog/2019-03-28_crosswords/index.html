A crossword puzzle editor

I've recently gotten excited about crosswords. I picked up my "code something up on a single flight" game from a while back and made a crossword editor. It does stuff the the symmetry rules automatically, which is fun I guess. If I wind up using it I'll finish the sharing feature, which I think is the one thing that makes this less useful than it could be.

Be careful when resizing, it completely wipes the board.
